Getting started

Dependencies

This repository contains a set of native AngularJS + Famo.us directives based on Ionic CSS. As a result no dependency on jQuery is required. The only required dependencies are:

  • AngularJS (requires AngularJS 1.3.x, tested with 1.3.10)
  • Famo.us (tested with version 0.3.4).

Slidebox (yoobic.us.slidebox)

The slidebox directive builds on top of the angular famous scroll view directive to provide a list of items, with scrolling capabilities and various animations.

The body of each slide is transcluded in to the body of the scrollable element.

            slideBoxDelegate.$getByHandle('myslidebox').goToPage(index);
